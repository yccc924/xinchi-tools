# 二手機圖片自動合成工具

## 字體設定

只需將字體檔案命名為 `font.ttf`，放入 `assets/` 資料夾即可。

> 例：將「王漢宗顏楷體繁.ttf」複製一份並改名為 `font.ttf` 放入 `assets/`。

---

## 安裝相依套件

```bash
pip install Pillow
```

---

## 使用方式

1. 將底圖（模板）放入 `input/` 資料夾（支援 `.jpg` `.jpeg` `.png` `.webp`）
2. 執行主程式：
   ```bash
   python main.py
   ```
3. 程式依序處理每張圖，每張圖詢問一次資訊：
   - 店保資訊（例：6個月）
   - 電池百分比（只輸入數字，例：85）
   - 顏色（例：黑色）
   - 編號（例：A001）
   - 型號（例：iPhone 15 Pro）
   - 容量（例：256GB）
4. 完成後，輸出圖片存至 `output/` 資料夾，檔名與原圖相同。

---

## 目錄結構

```
assets/
  font.ttf          ← 字體放這裡
input/              ← 放入原始底圖
output/             ← 合成後的圖輸出於此
image_engine.py
main.py
README.md
```

## 版面說明

| 區域 | 位置 | 內容 |
|------|------|------|
| 上方資訊列 | 圖片上方約 14% 處 | 店保 / 電池 / 顏色 / 編號 |
| 底部型號條 | 圖片最底部 | 橘色邊框黑色長條，型號與容量置中 |

所有文字皆使用**雙層黑色描邊**，確保在任何背景下清晰可見。
